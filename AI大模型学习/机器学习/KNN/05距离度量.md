# 距离度量（Distance Measure）

## 第一部分：距离度量的基本概念

### 1.1 为什么需要距离度量

在 KNN 算法中，判断两个样本是否相似的核心依据是它们在**特征空间中的距离**。距离越小，表示样本越相似。距离度量直接影响 KNN 的分类/回归结果。

---

### 1.2 距离度量的数学性质（度量公理）

一个函数 $d\left(a,b\right)$ 要被称为“距离”，通常需满足以下四条公理：

| 性质    | 数学表达                                                       | 含义                   |
| ----- | ---------------------------------------------------------- | -------------------- |
| 非负性   | $d\left(a,b\right)\ge0$                                    | 距离不能为负数              |
| 同一性   | $d\left(a,b\right)=0\iff a=b$                              | 只有同一个点距离为 0          |
| 对称性   | $d\left(a,b\right)=d\left(b,a\right)$                      | 从 a 到 b 的距离等于从 b 到 a |
| 三角不等式 | $d\left(a,c\right)\le d\left(a,b\right)+d\left(b,c\right)$ | 两边之和大于第三边            |

> 下面四种距离均满足这些性质。

---

### 1.3 通用符号定义

- 两个 $n$ 维向量（样本）：

$$a = (x_11,x_12,...,x_n)$$

$$b = (x_21,x_22,...,x_{2n})$$

- 下标含义：第一个下标表示样本编号，第二个下标表示特征难度。

---

## 第二部分：欧氏距离（Euclidean Distance）

### 2.1 定义与公式

> 直观的距离度量方法，两个点在空间中的距离一般都是指欧氏距离。

**标准公式**（$n$ 维）：

$$d_{euclidean}\left(a,b\right)=\sqrt{\sum_{i=1}^{n}\left(x_{1i}-x_{2i}\right)^2}$$

**二维特例**（平面两点）：

$$d=\sqrt{\left(x_1-x_2\right)^2+\left(y_1-y_2\right)^2}$$

**三维特例**：

$$d=\sqrt{\left(x_1-x_2\right)^2+\left(y_1-y_2\right)^2+\left(z_1-z_2\right)^2}$$

---

### 2.2 几何意义

- 对应欧几里得几何中的**直线距离**。
- 二维平面上，是两点间线段的长度。

---

### 2.3 直观图示

在二维坐标系中，点 $P(1,1)$ 和 $Q(4,5)$ 的欧氏距离为：

$$\sqrt{\left(4-1\right)^2+\left(5-1\right)^2}=\sqrt{9+16}=5$$

即点 P 到点 Q 的直线距离为 5.

---

### 2.4 代码实现

```python
import numpy as np

# 手动实现
def euclidean_distance(a,b):
	return np.sqrt(np.sum((a - b) ** 2))

# 使用scipy
from scipy.spatial.distance import euclidean

# 使用sklearn
from sklearn.metrics.pairwise import euclidean_distances

a = np.array([1,1])
b = np.array([4,5])
print(euclidean_distance(a,b)) # 5.0
```

---

### 2.5 优缺点

| 优点       | 缺点              |
| -------- | --------------- |
| 直观易懂，最常用 | 对特征量纲敏感（需要标准化）  |
| 数学性质良好   | 高维空间中效果下降（维度灾难） |
| 旋转不变性    | 各维度独立假设（不考虑相关性） |

---

### 2.6 在 KNN 中的应用

KNN 默认使用欧式距离（`metric = 'euclidean'` 或 `p=2`）。

## 第三部分：曼哈顿距离（Manhatan Distance）

### 3.1 定义与公式

> 也称为“城市街区距离”（City Block distance），曼哈顿城市特点：横平竖直。

**公式**：

$$d_{manhattan}\left(a,b\right)=\sum_{i=1}^{n}\left|x_{1i}-x_{2i}\right|$$

即各维度绝对差值的总和。

---

### 3.2 几何意义

- 在只能沿坐标轴方向移动（不能走斜线）的情况下，从一点到另一点所需的最短路径长度。
- 名字来源于纽约曼哈顿的网格状街道布局：从一个十字路口到另一个十字路口，只能横着或竖着走，走过的总街区就是曼哈顿距离。

**二维示例**：

- 从 $(1,1)$ 到 $(4,5)$
- 路线 1：先向右 3 格，在向上 4 格
	- 3+4=7
- 路线 2：先向上 4 格，再向右 3 格
	- 4+3=7
- 任何合法路径总长度均为 7.

---

### 3.3 公式对比（与欧氏距离）

对于二维点 $\left(x_1,y_1\right)$ 和 $\left(x_2,y_2\right)$：

- 欧氏距离：$\sqrt{\left(\Delta x\right)^2+\left(\Delta y\right)^2}$
- 曼哈顿距离：$\left|\Delta x\right|+\left|\Delta y\right|$
当 $\Delta x$ 和 $\Delta y$ 都为正时，曼哈顿距离 $\ge$ 欧氏距离（等号仅在其中一个 $\Delta$ 为 0 时成立）。

---

### 3.4 代码实现

```python
def manhattan_distance(a,b):
	return np.sum(np.abs(a - b))

# scipy版本
from scipy.spatial.distance import cityblock
print(cityblock([1,1],[4,5])) # 7
```

---

### 3.5 优缺点

| 优点             | 缺点                |
| -------------- | ----------------- |
| 计算简单（无平方根）     | 对坐标轴方向敏感（旋转后距离会变） |
| 对异常值的敏感度低于欧氏距离 | 在高维空间中表现不一定优于欧氏距离 |
| 适合网格状结构的数据     | -                 |

---

### 3.6 何时使用曼哈顿距离

- 特征之间相互独立、且每个特征的差异同等重要时。
- 数据具有“网格”或“街区”结构（如城市路径规划、棋盘移动）。
- 希望强调绝对差异而非平方差异（降低大差异的权重）。

## 第四部分：切比雪夫距离（Chebyshev Distance）

### 4.1 定义与公式

> 国际象棋中，国王可以直行、横行、斜行，所以国王走一步可以移动到相邻 8 个方格中的任意一个。国王从格子(x 1,y 1)走到格子(x 2,y 2)最少需要多少步？这个距离就叫切比雪夫距离。

**公式**：

$$d_{chebyshev}\left(a,b\right)=\max_{i}\left|x_{1i}-x_{2i}\right|$$

即各维度绝对差值的**最大值**。

---

### 4.2 几何意义

- 棋盘上国王移动的最少步数（国王可以沿 8 个方向任意移动，包括对角线）。
- 在二维中，切比雪夫距离等于 $\max\left(\left|\Delta x\right|,\left|\Delta y\right|\right)$。

**示例**：

- 从 $\left(1,1\right)$ 到 $\left(4,5\right)$
- $\left|\Delta x\right|=3,\left|\Delta y\right|=4,\max=4$。国王只需 4 步（例如先斜走 3 步到 $\left(4,4\right)$，再直走 1 步到 $\left(4,5\right)$）。

---

### 4.3 与其他距离的关系

| 距离   | 二维表达式                          |                                                  |
| ---- | ------------------------------ | ------------------------------------------------ |
| 欧氏   | $\sqrt{\Delta x^2+\Delta y^2}$ |                                                  |
| 曼哈顿  | $\left\vert\Delta x\right\vert+\left\vert\Delta y\right\vert$        |
| 切比雪夫 | $\max\left(\left\vert\Delta x\right\vert+\left\vert\Delta y\right\vert\right)$ |

对于二维，三者大小关系：

切比雪夫 $\le$ 欧氏 $\le$ 曼哈顿（当 $\Delta x,\Delta y>0$ 时，不一定严格，需注意：切比雪夫可能大于欧氏）

---

### 4.4 代码实现

```python
def chebyshev_distance(a, b):
    return np.max(np.abs(a - b))

# scipy
from scipy.spatial.distance import chebyshev
print(chebyshev([1,1], [4,5]))  # 4
```

---

### 4.5 扩展：棋盘上的其他棋子

- **车**（Rook）：只能直行横行
	- 曼哈顿距离
- **王**（King）：八方移动
	- 切比雪夫距离
- **后**（Queen）：八方移动
	- 切比雪夫距离

---

### 4.6 应用场景

- 棋盘游戏 AI（如国际象棋、围棋中的某些距离计算）。
- 仓库中机器人的最短路径（允许对角线移动时）。
- 某些图像处理中距离变换。

## 第五部分：闵可夫斯基距离（Minkowski Distance）

### 5.1 定义与公式

> 闵氏距离不是一种新的距离度量方式，而是对多个距离度量公式的概括性表述。

**公式**：

$$d_{minkowski}\left(a,b\right)=\left(\sum_{i=1}^{n}\left|x_{1i}-x_{2i}\right|^{p}\right)^{\frac{1}{p}}$$
或（公式变换）：
$$d_{minkowski}\left(a,b\right)=\sqrt[p]{\sum_{i=1}^{n}\left|x_{1i}-x_{2i}\right|^{p}}$$

其中参数 $p\ge1$。

---

### 5.2 参数 p 与常见距离的关系

| p 值          | 对应距离   | 说明                                                                                                                                         |     |
| ------------ | ------ | ------------------------------------------------------------------------------------------------------------------------------------------ | --- |
| $p=1$        | 曼哈顿距离  | $\left(\sum\left\vert\Delta x_{i}\right\vert^1\right)^1=\sum\left\vert\Delta x_{i}\right\vert$                                             |     |
| $p=2$        | 欧氏距离   | $\sqrt{\sum\left\vert\Delta x_{i}\right\vert^2}$                                                                                           |     |
| $p\to\infty$ | 切比雪夫距离 | $\lim_{p\to\infty}\left(\sum\left\vert\Delta x_{i}\right.\left\vert\right.^{p}\right)^{\frac{1}{p}}=\max\left\vert\Delta x_{i}\right\vert$ |     |

**中间值**（如 p=3，p=4 等）没有专门的名称，但可用于控制距离对大差异的敏感程度：p 越大，越重视最大的差异。

----

### 5.3 几何意义

- 不同 p 值对应不同的“范数”（Lp 范数）。
- 在二维平面上，到原点距离为 1 的点组成的形状：
	- $p=1$（曼哈顿）：菱形（正方形旋转 45°）。
	- $p=2$ （欧氏）：圆形。
	- $p=\infty$（切比雪夫）：正方形（边平行于坐标轴）。
	- 中间 p 值：介于菱形和圆形之间的“圆角方形”。

---

### 5.4 代码实现

```python
def minkowski_distance(a, b, p):
    return np.power(np.sum(np.abs(a - b) ** p), 1/p)

# scipy
from scipy.spatial.distance import minkowski
print(minkowski([1,1], [4,5], p=3))
```

---

### 5.5 优点与注意事项

| 优点                                          | 注意事项                    |
| ------------------------------------------- | ----------------------- |
| 统一框架                                        | $p<1$ 时不满足三角不等式（不是真正距离） |
| 可通过调整 p 控制对异常值的敏感度                          | p 太大时计算可能溢出（需要归一化）      |
| sklearn 的 KNN 默认使用 `p=2，metric='minkowski'` | -                       |

---

### 5.6 如何选择 p 值

- $p=1$：特征差异独立、无相关性，希望鲁棒性更好时。
- $p=2$：默认选择，效果通常不错。
- $p>2$：希望强调最大差异方向的特征。
- 实践中常通过交叉验证选择最优的 p 和 K。

## 第六部分：距离度量在 KNN 中的综合对比

| 距离    | 公式                                                               | 对差异度敏感度   | 计算复杂度 | 旋转不变性    | 适用场景       |
| ----- | ---------------------------------------------------------------- | --------- | ----- | -------- | ---------- |
| 欧氏    | $\sqrt{\sum\Delta^2}$                                            | 高（平方放大）   | O(n)  | 是        | 通用默认       |
| 曼哈顿   | $\sum\left\vert\Delta\right\vert$                                | 中         | O(n)  | 否        | 网络数据、独立特征  |
| 切比雪夫  | $\max\left\vert\Delta\right\vert$                                | 低（只关注最大）  | O(n)  | 否        | 棋盘移动、允许对角线 |
| 闵可夫斯基 | $\left(\sum\left\vert\Delta\right\vert^{p}\right)^{\frac{1}{p}}$ | 随 p 增大而降低 | O(n)  | $p=2$ 时是 | 通过调参灵活适配   |

## 第七部分：扩展知识

### 7.1 其他常见距离

| 距离    | 公式                                                                                | 用途                |     |
| ----- | --------------------------------------------------------------------------------- | ----------------- | --- |
| 余弦相似度 | $1-\frac{\left(a\cdot b\right)}{\left\vert a\right\vert \left\vert b\right\vert}$ | 文本相似度（忽略绝对大小）     |     |
| 马氏距离  | $\sqrt{\left(a-b\right)^{T}S^{-1}\left(a-b\right)}$                               | 考虑特征相关性（S 为协方差矩阵） |     |
| 汉明距离  | 连个等长字符串对应位置不同字符的个数                                                                | 信息论、编码理论          |     |
| 杰卡德距离 | $1-\frac{A\cap B}{A\cup B}$                                                       | 几何相似度             |     |

---

### 7.3 维度灾难与距离度量

高维空间中，所有点对之间的距离分布趋于集中（最远距离与最近距离的比值趋近于 1），导致 KNN 失效。

缓解方法：

- 特征降维（PCA、t-SNE）。
- 使用更鲁棒的距离（如马氏距离）。
- 增大 K 值（但效果有限）。
