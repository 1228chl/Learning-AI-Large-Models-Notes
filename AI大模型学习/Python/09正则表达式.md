## 一、详细笔记

### 第1部分：正则表达式概述

#### 1.1 为什么要学习正则表达式

在实际开发中，经常需要查找符合某些规则的字符串，例如：

- 判断用户输入的邮箱格式是否正确
- 从网页源码中提取图片地址
- 验证手机号码是否合法
- 过滤敏感词

使用正则表达式可以高效、简洁地完成这些任务。

#### 1.2 什么是正则表达式

- **定义**：正则表达式（regular expression）是一种描述字符串匹配模式的特殊语法。它通过一些特殊的符号组成一个模式，用于检索、替换或提取符合该模式的子串。
- **功能**：
  1. **检索**：判断一个字符串是否含有某种子串。
  2. **替换**：将匹配的子串替换为其他内容。
  3. **提取**：从字符串中取出符合某个条件的子串。
- **跨语言性**：正则表达式并不是 Python 特有的，在 Java、PHP、Go、JavaScript 等语言中也都支持。学会正则表达式，可以在多种语言中使用。

#### 1.3 正则表达式的应用场景

| 场景                 | 示例                             |
| -------------------- | -------------------------------- |
| 数据验证（表单验证） | 手机号、邮箱、IP地址、身份证号等 |
| 数据检索（爬虫）     | 从网页中提取链接、图片、标题等   |
| 数据隐藏             | 手机号中间四位隐藏为 `****`      |
| 数据过滤             | 论坛评论中的敏感词替换为 `***`   |

> **特点**：正则表达式语法令人头疼，可读性较差；但通用性很强，适用于很多编程语言。

### 第2部分：re 模块介绍

在 Python 中使用正则表达式，需要导入内置的 `re` 模块。

#### 2.1 re 模块使用三步走

```python
import re

# 第一步：使用匹配方法得到匹配对象
result = re.match(pattern, string, flags=0)

# 第二步：如果匹配成功，使用 group() 方法提取数据
if result:
    data = result.group()
```

#### 2.2 常用匹配方法

##### 2.2.1 re.match()

- **语法**：`re.match(pattern, string, flags=0)`
- **功能**：尝试**从字符串的起始位置**匹配正则表达式。如果起始位置不匹配，则返回 `None`；如果匹配成功，返回一个匹配对象。
- **注意**：`match` 不会扫描整个字符串，只从开头匹配。

```python
result = re.match(".it.", "aitcast")   # 匹配成功，返回 "aitc"
result = re.match(".it.", "iloveitcast")  # 匹配失败，返回 None
```

##### 2.2.2 re.search()

- **语法**：`re.search(pattern, string, flags=0)`
- **功能**：**扫描整个字符串**，返回第一个成功的匹配。如果找不到任何匹配，返回 `None`。

```python
result = re.search("\d.*", "city:1beijing2.shanghai")
print(result.group())   # 输出：1beijing2.shanghai
```

##### 2.2.3 re.findall()

- **语法**：`re.findall(pattern, string, flags=0)`
- **功能**：**扫描整个字符串**，返回所有成功匹配的子串，以**列表**形式返回。如果没有任何匹配，返回空列表 `[]`。

```python
result = re.findall('itheima', 'www.itheimaitheima.com')
print(result)   # ['itheima', 'itheima']
```

##### 2.2.4 替换文本：re.sub()

- **语法**：`re.sub(pattern, string, flags=0)`
- 功能：**扫描整个字符串**，替换所有匹配到的字符串，返回替换后的**新字符串**（原字符串不变）。如果找不到任何匹配，返回**原字符串**。

```Python
result = re.sub('itheima', 'hell', 'www.itheimaitheima.com')
print(result)    #www.hellhell.com
```

#### 2.3 提取匹配数据：group() 方法

- 对于 `match` 和 `search` 返回的匹配对象，可以使用 `group()` 方法获取匹配到的完整子串。

```Python
res = re.search(r'\d+', 'abc123def')

print(res)       # 打印对象本身
print(res.group())# 取出匹配内容
```

- `group(0)` 或 `group()` 表示整个匹配；`group(1)`、`group(2)` 表示第1、2个分组（后面会讲）。
- 无参数：取全部匹配内容

```Python
res.group()
```

- 带数字：提取括号分组（最重要！）

```Python
res = re.search(r'(\d+)-(\w+)', 'id:10086-hello')

print(res.group())      # 全部：10086-hello
print(res.group(1))     # 第1组：10086
print(res.group(2))     # 第2组：hello
```

- groups ()：一次性取出所有分组，返回元组

```Python
res.groups()  # ('10086', 'hello')
```

### 第3部分：正则表达式编写（核心语法）

#### 3.1 匹配单个字符

| 元字符 | 说明                                       | 等价于          | 示例                                                     |
| ------ | ------------------------------------------ | --------------- | -------------------------------------------------------- |
| `.`    | 匹配任意1个字符（除了换行符 `\n`）         | —               | `"a.c"` 可匹配 `"abc"`、`"a1c"`                          |
| `[ ]`  | 匹配中括号中列举的字符                     | —               | `[abc]` 匹配 `a` 或 `b` 或 `c`；`[a-z]` 匹配任意小写字母 |
| `\d`   | 匹配数字                                   | `[0-9]`         | `"itcast\d"` 可匹配 `"itcast5"`                          |
| `\D`   | 匹配非数字                                 | `[^0-9]`        | `"itcast\D"` 可匹配 `"itcast-"`                          |
| `\s`   | 匹配空白（空格、`\t`、`\n`等）             | —               | `"itcast\s111"` 匹配 `"itcast\t111"`                     |
| `\S`   | 匹配非空白                                 | —               | `"itcast\S"` 不匹配 `"itcast\t"`                         |
| `\w`   | 匹配非特殊字符：字母、数字、下划线、汉字   | `[a-zA-Z0-9_]`  | `"itcast\w"` 匹配 `"itcast_"`、`"itcasta"`               |
| `\W`   | 匹配特殊字符（非字母、数字、下划线、汉字） | `[^a-zA-Z0-9_]` | `"itcast\W"` 匹配 `"itcast\t"`、`"itcast!"`              |

**注意**：`\w` 在 Python 3 中默认也匹配汉字（Unicode 字母），但通常只用于 ASCII 字母数字下划线。

#### 3.2 匹配多个字符（量词）

| 量词    | 说明                                                     | 示例                                                         |
| ------- | -------------------------------------------------------- | ------------------------------------------------------------ |
| `*`     | 匹配前一个字符出现 **0 次或无限次**（可有可无）          | `"itcast1*"` 匹配 `"itcast"`、`"itcast111"`                  |
| `+`     | 匹配前一个字符出现 **1 次或无限次**（至少1次）           | `"itcast\d+itcast"` 匹配 `"itcast123itcast"`                 |
| `?`     | 匹配前一个字符出现 **0 次或 1 次**                       | `"itcast\d?itcast"` 匹配 `"itcastitcast"` 或 `"itcast1itcast"` |
| `{m}`   | 匹配前一个字符出现 **m 次**                              | `"\d{2}"` 匹配连续2个数字                                    |
| `{m,n}` | 匹配前一个字符出现 **m 到 n 次**（注意逗号后不能有空格） | `"\d{2,5}"` 匹配连续2~5个数字                                |

**示例**：

```python
re.match("itcast\d*itcast", "itcast123itcast")   # 匹配
re.match("itcast\d+itcast", "itcastitcast")      # 不匹配（数字至少1次）
re.match("itcast\d{2,5}itcast", "itcast111itcast") # 匹配
```

#### 3.3 匹配开头和结尾

| 元字符        | 说明                                                         | 示例                                               |
| ------------- | ------------------------------------------------------------ | -------------------------------------------------- |
| `^`           | 匹配字符串**开头**                                           | `"^\ditcast"` 匹配以数字开头后跟 `itcast` 的字符串 |
| `$`           | 匹配字符串**结尾**                                           | `".*\d$"` 匹配以数字结尾的字符串                   |
| `[^指定字符]` | 匹配**除了**指定字符以外的所有字符（注意：放在 `[]` 内表示取反） | `"[^4]$"` 表示结尾不能是字符 `4`                   |

**组合使用**：

```python
re.match("^\d.*\d$", "11itcast22")   # 以数字开头，以数字结尾
re.match("^\d.*[^4]$", "11itcast@")  # 结尾不是4
```

#### 3.4 匹配分组

| 符号           | 说明                 | 示例                                            |
| ------------ | ------------------ | --------------------------------------------- |
| `\|`         | 匹配左右任意一个表达式（或）     | `"apple\|pear"` 匹配 `"apple"` 或 `"pear"        |
| `( )`        | 将括号中的字符作为一个**分组**  | `"(qq):([1-9]\d{4,11})"` 将 `qq` 作为分组1，号码作为分组2 |
| `\num`       | 引用第 num 个分组匹配到的字符串 | `"<(w+)>.*</1>"` 匹配成对标签                       |
| `(?P<name>)` | 给分组起别名             | `"<(?P<tag>w+)>.*</(?P=tag)>"`                |
| `(?P=name)`  | 引用别名为 name 的分组     | 同上                                            |

**示例1：使用 `|` 匹配多个水果**

```python
fruit = ["apple", "banana", "orange", "pear"]
for value in fruit:
    result = re.match("apple|pear", value)
    if result:
        print("我想吃的水果:", value)
```

**示例2：匹配邮箱（163、126、qq）**

```python
# 注意：. 需要转义为 \.
result = re.match("[a-zA-Z0-9_]{4,20}@(163|126|qq)\.com", "hello@qq.com")
```

**示例3：提取 qq 号和类型**

```python
result = re.match("(qq):([1-9]\d{4,11})", "qq:10567")
if result:
    print(result.group(0))   # qq:10567
    print(result.group(1))   # qq
    print(result.group(2))   # 10567
```

**示例4：匹配成对标签（使用分组引用）**

```python
# 方式1：使用 \num
result = re.match(r"<([a-zA-Z1-6]{4})>.*</\1>", "<html>hh</html>")
# 方式2：使用命名分组
result = re.match(r"<(?P<name1>[a-zA-Z1-6]{4})>.*</(?P=name1)>", "<html>hh</html>")
```

> **注意**：使用 `\num` 时，如果正则表达式字符串中包含反斜杠，建议使用**原始字符串** `r"..."`，避免转义问题。

### 第4部分：贪婪模式与非贪婪模式

#### 4.1 概念

- **贪婪模式（Greedy）**：在整个表达式匹配成功的前提下，**尽可能多地**匹配字符。量词 `*`、`+`、`{m,n}` 默认是贪婪的。
- **非贪婪模式（Non-greedy）**：在整个表达式匹配成功的前提下，**尽可能少地**匹配字符。在量词后面加 `?` 即可变为非贪婪模式：`*?`、`+?`、`??`、`{m,n}?`。

#### 4.2 示例对比

```python
import re
my_str = '<h1>编程语言</h1>此处省略很多内容<h1>python大数据</h1>'

# 贪婪模式：匹配尽可能多的内容
data = re.findall('<h1>.*</h1>', my_str)
print(data)   # ['<h1>编程语言</h1>此处省略很多内容<h1>python大数据</h1>']

# 非贪婪模式：匹配尽可能少的内容
data = re.findall('<h1>.*?</h1>', my_str)
print(data)   # ['<h1>编程语言</h1>', '<h1>python大数据</h1>']

# 提取标签内的标题（使用非贪婪 + 分组）
data = re.findall('<h1>(.*?)</h1>', my_str)
print(data)   # ['编程语言', 'python大数据']
```

#### 4.3 应用场景

- 提取 HTML/XML 标签内容时，必须使用非贪婪模式，否则会匹配到第一个 `<h1>` 到最后一个 `</h1>` 之间的所有内容。
- 在爬虫开发中，非贪婪模式非常常用。

### 第5部分：正则表达式修饰符（标志位）

修饰符用于改变正则表达式的匹配行为。在 `re.match`、`re.search`、`re.findall` 中通过 `flags` 参数指定。

| 修饰符    | 说明                             | 示例                                         |
| ------ | ------------------------------ | ------------------------------------------ |
| `re.I` | 匹配时**不区分大小写**                  | `re.findall('^a8D9$', 'A8d9', re.I)` 能匹配成功 |
| `re.S` | 使 `.` 匹配包括**换行符 `\n`** 在内的所有字符 | 默认 `.` 不匹配换行，使用 `re.S` 后可匹配                |
| `re.M` | 匹配多行模式                         | `^`匹配每一行开头，`$`匹配每一行结尾。                     |
| 多个标志组合 | 使用按位或 `\|`                     | `re.S \| re.I`                             |

#### 5.1 re.I(ignoreCase):忽略大小写

- 场景：匹配时不区分A/a,B/b

```Python
import re
text = "Apple apple APPLE"

# 不忽略：只能匹配 Apple
print(re.findall(r"apple", text))  # ['apple']

# 忽略大小写：匹配全部
print(re.findall(r"apple", text, re.I))  # ['Apple', 'apple', 'APPLE']
```

---

#### 5.2 re.S:匹配换行

- 场景：默认`.`不匹配换行`\n`，使用后可以检测到一切字符，用于爬虫提取文本时用

```Python
text = "hello\nworld"

# 默认：匹配不到
print(re.findall(r"hello.*world", text))  # []

# 加 re.S：. 可以匹配换行
print(re.findall(r"hello.*world", text, re.S))  # ['hello\nworld']
```

---

#### 5.3 re.M:多行模式

- 场景：让 `^` 匹配**每一行开头**，`$` 匹配**每一行结尾**。

```Python
text = "first line\nsecond line\nthird line"

# 不加 re.M：^ 只匹配整个字符串开头
print(re.findall(r"^.*line", text))  # ['first line']

# 加 re.M：^ 匹配每一行开头
print(re.findall(r"^.*line", text, re.M))
# ['first line', 'second line', 'third line']
```

import re
fruit = ["apple", "banana", "orange", "pear"]

html = "<html><h1>www.itcast.cn</h1></html>"

## 三、常见面试/考试题

1. **`re.match` 和 `re.search` 的区别？**  
   `match` 只从字符串开头匹配；`search` 扫描整个字符串，返回第一个匹配。

2. **`re.findall` 返回什么类型？**  
   返回列表，没有匹配时返回空列表。

3. **如何匹配任意字符（包括换行）？**  
   使用 `re.S` 修饰符，让 `.` 匹配换行符；或者使用 `[\s\S]` 等组合。

4. **什么是贪婪模式？如何改为非贪婪？**  
   贪婪模式是尽可能多匹配；在量词后加 `?` 改为非贪婪，如 `*?`、`+?`。

5. **如何提取分组中的数据？**  
   使用 `group(1)`、`group(2)`... 或使用命名分组 `group('name')`。

6. **如何匹配一个合法的邮箱地址？**  
   简单版：`\w+@\w+\.\w+`；更精确版需要根据具体规则。

7. **如何匹配成对出现的 HTML 标签？**  
   使用分组引用，如 `r"<(\w+)>.*?</\1>"`（结合非贪婪）。
